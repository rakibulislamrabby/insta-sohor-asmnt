How does Javascript Work

Javascript হচ্ছে একটি interpreted language। এর মানে ব্রাউজারে পাঠানোর আগে আমাদের জাভাস্ক্রিপ্ট সোর্স কোড কম্পাইল করতে হবে না। যখনই আমরা একটি ওয়েব ব্রাউজারের ভিতরে একটি জাভাস্ক্রিপ্ট প্রোগ্রাম চালাই, তখন ব্রাউজারের engine জাভাস্ক্রিপ্ট কোডটি গ্রহণ করে এবং engine সোর্স কোডটি output করে দেই। Javascript এ একটি শক্তিশালী engine আছে সেইটার নাম হচ্ছে “V8” Javascript আবার dynamically typed language. এটা C অথবা C++ এর মত না। Javascript এ আমরা var, let, const দিয়ে ভ্যারিয়েবল ডিক্লার করতে পারি এখন বেশি ব্যাবহার হয় let এবং const. Let দিয়ে ডিক্লার করা ভ্যারিয়েবলের ভ্যালু চেঞ্জ করা যায়। কিন্তু const দিয়ে ডিক্লার করলে সেইটার ভ্যালু আর পরিবর্তন করা যায় না। প্রথমে ওয়েব পেজ তৈরি করার জন্য এইচটিএমএল এবং সিএসএসকে DOM দ্বারা একত্রিত করা হয়।তারপর, ব্রাউজারের জাভাস্ক্রিপ্ট ইঞ্জিন জাভাস্ক্রিপ্ট ফাইল এবং ইনলাইন কোড লোড করে কিন্তু কোডটি সাথে সাথে রান করে নাহ।এটি HTML এবং CSS Load শেষ হওয়ার জন্য অপেক্ষা করে।এবং এটি হয়ে যাওয়ার পর Javascript executed করে। এর ফলে DOM জাভাস্ক্রিপ্ট কোড দ্বারা আপডেট হয় এবং ব্রাউজার দ্বারা তা render করা হয়।
